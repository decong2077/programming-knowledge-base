    使用指南¶

    新手入门¶

    1. 从 编程导论 开始
    2. 按照 编程学习路径 学习
    3. 选择一门语言深入 (Python 推荐)
    4. 实践项目巩固知识

    进阶提升¶

    1. 学习 设计模式
    2. 掌握 测试 方法
    3. 参与 技术社区
    4. 阅读 编程书籍推荐

    求职准备¶

    1. 刷算法题 (技术面试准备)
    2. 准备项目经验
    3. 系统设计练习
    4. 模拟面试
