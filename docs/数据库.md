# 数据库

## 概述
数据库是组织、存储和管理数据的系统。

## 数据库类型

### 关系型数据库 (SQL)
- **MySQL** - 最流行的开源数据库
- **PostgreSQL** - 功能强大的对象关系数据库
- **SQLite** - 轻量级嵌入式数据库

### 非关系型数据库 (NoSQL)
- **MongoDB** - 文档数据库
- **Redis** - 内存键值数据库 (缓存首选)
- **Elasticsearch** - 搜索引擎数据库

## 核心概念
- 表/集合设计
- 索引优化
- 查询优化
- 事务处理
- 数据备份与恢复

## 设计原则
1. 规范化设计 (减少冗余)
2. 适当反规范化 (提升性能)
3. 合理设计索引
4. 考虑数据增长

## 性能优化

- 索引策略
- 缓存层 (Redis)
- 读写分离
- 分库分表
