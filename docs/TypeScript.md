# TypeScript

## 概述
TypeScript 是 JavaScript 的超集,添加了类型系统。

## 为什么用 TypeScript
- **类型安全** - 编译时发现错误
- **更好的 IDE 支持** - 智能提示、重构
- **自文档化** - 类型即文档
- **大型项目友好** - 可维护性强
