# 编程书籍推荐

## 入门必读
### 1. 代码大全 (Code Complete)
- 作者: Steve McConnell
- 适合: 所有阶段
- 内容: 软件构建的完整指南

### 2. 程序员修炼之道 (The Pragmatic Programmer)
- 作者: David Thomas, Andrew Hunt
- 适合: 初级到中级
- 内容: 编程哲学和最佳实践

### 3. Clean Code (代码整洁之道)
- 作者: Robert C. Martin
- 适合: 有一定经验后
- 内容: 如何写可读的代码
- 关联: [代码规范](代码规范.md)

## 语言专项

### Python
- Fluent Python - 进阶必读
- Python Cookbook - 实用技巧集

### JavaScript
- JavaScript 高级程序设计 - 入门到进阶
- You Don't Know JS - 深入理解核心概念
- Effective JavaScript - 最佳实践

### Go
- Go 语言圣经 - 系统学习
- Go 程序设计语言 - 权威指南
