# 后端开发

## 概述
后端开发处理服务器逻辑、数据库交互和 API 设计。

## 主流技术栈

### Python 后端
- 框架: FastAPI, Django, Flask
- 优势: 开发快速,生态丰富
- 场景: 快速原型,数据分析服务
- 参考: [Python](Python.md)

### Node.js 后端
- 框架: Express, Koa, NestJS
- 优势: 前后端语言统一
- 场景: I/O 密集型应用
- 参考: [JavaScript](JavaScript.md)

### Go 后端
- 框架: Gin, Echo
- 优势: 高性能,原生并发
- 场景: 高并发,微服务
- 参考: [Go](Go.md)

## 核心概念
- RESTful API 设计
- 认证与授权 (JWT, OAuth)
- 中间件机制
- 错误处理
- 日志系统

## 数据库集成
- ORM: SQLAlchemy, Prisma, GORM
- 参见 [数据库](数据库.md)

## 架构模式
- MVC 架构
- 微服务架构
- 事件驱动架构
