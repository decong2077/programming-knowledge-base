# 前端开发

## 概述
前端开发关注用户界面和交互体验,是用户与产品的第一触点。

## 核心技术
### 三剑客
1. **HTML** - 页面结构
2. **CSS** - 样式和布局
3. **[JavaScript](JavaScript.md)** - 交互逻辑

### 现代前端
- **[TypeScript](TypeScript.md)** - 类型安全
- **框架**: React, Vue, Angular
- **构建工具**: Vite, Webpack
- **包管理**: npm, yarn, pnpm

## React 生态
- React 18 + Hooks
- Next.js (SSR/SSG)
- 状态管理: Zustand, Redux
- UI 库: Ant Design, Material-UI

## Vue 生态
- Vue 3 + Composition API
- Nuxt.js (SSR)
- 状态管理: Pinia
- UI 库: Element Plus

## 核心技能
- 响应式设计
- CSS 预处理器 (Sass/Tailwind)
- 组件化开发
- 性能优化
- 浏览器调试

## 学习路径
1. HTML/CSS 基础
2. [JavaScript](JavaScript.md) 核心
3. 框架学习 (React 或 Vue)
4. 工程化: [版本控制](版本控制.md) + [测试](测试.md)
5. 性能优化进阶

参考: [编程学习路径](编程学习路径.md)
