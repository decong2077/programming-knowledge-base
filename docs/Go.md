# Go 语言

## 概述
Go (Golang) 是 Google 开发的编译型语言,专注于简洁、高效、并发。

## 核心特性
- **简洁语法** - 学习曲线平缓
- **原生并发** - goroutine 和 channel
- **快速编译** - 编译速度极快
- **静态类型** - 类型安全
- **内置工具** - 测试、格式化、依赖管理

## 应用领域
- **云原生**: Docker, Kubernetes 都是用 Go 写的
- **微服务**: gRPC, Gin 框架
- **[[后端开发]]**: 高并发 API 服务
- **[[DevOps]]**: 工具链开发
- **网络编程**: 高性能网络服务

## 优势场景
- 高并发场景 (相比 [[Python]] 性能更好)
- 需要快速启动和编译
- 团队项目 (代码风格统一)
- 云原生项目

## 学习路径
1. 基础语法 (1-2 周)
2. 标准库使用
3. 并发编程
4. Web 框架 (Gin/Echo)
5. 微服务架构
