# 编程学习路径

## 阶段一：编程基础 (1-2个月)
### 核心内容
- 学习 [[Python]] 基础语法
- 理解变量、循环、函数
- 掌握基本数据结构

### 实践项目
- 计算器程序
- 待办事项列表
- 简单爬虫

参考: [[编程导论]]

## 阶段二：Web 开发 (2-3个月)
### 前端基础
- HTML/CSS 基础
- [[JavaScript]] 核心概念
- DOM 操作

### 后端入门
- [[Python]] Flask/FastAPI
- 基础 [[数据库]] 操作
- RESTful API 设计

参考: [[前端开发]]、[[后端开发]]

## 阶段三：工程化 (2-3个月)
### 核心技能
- [[TypeScript]] 类型系统
- [[版本控制]] Git 工作流
- [[代码规范]] 和重构
- [[测试]] 单元测试

### 工具链
- 包管理器 (npm/pip)
- 构建工具
- [[DevOps]] 基础

参考: [[工程实践]]

## 阶段四：进阶提升 (持续)
### 深入学习
- [[设计模式]]
- 系统架构
- 性能优化
- [[数据库]] 进阶

### 专业方向
选择 1-2 个方向深入:
- 前端专家: React/Vue 生态
- 后端专家: 微服务、分布式
- 全栈工程师
- DevOps 专家

参考: [[程序员成长路线]]

## 学习建议
1. **边学边练** - 每天至少 1 小时编码
2. **项目驱动** - 用实际项目巩固知识
3. **阅读源码** - 学习优秀代码
4. **写技术博客** - 输出倒逼输入
5. **参与开源** - 积累实战经验
