# 技术面试准备

- 图 (BFS, DFS, 最短路径)
- 排序算法 (快排、归并、堆排)
- 哈希表
- 动态规划
- 贪心算法

### 2. 系统设计
- 设计 URL 短链服务
- 设计 Twitter/微博
- 设计聊天系统
- 分布式系统设计
- [数据库](数据库.md) 设计

### 3. 项目经验
- 项目难点和解决
- 技术选型原因
- 架构演进过程
- 性能优化经验

### 4. 语言专项
- [Python](Python.md) 核心概念
- [JavaScript](JavaScript.md) 异步编程
- [Go](Go.md) 并发模型
- 框架原理理解

## 算法刷题

### 推荐平台
- LeetCode - 最常用
- HackerRank - 入门友好
- 牛客网 - 中文题库

### 刷题策略
1. 按类型刷 (不要随机)
2. 先理解思路,再写代码
3. 总结模板
4. 反复练习高频题

### 高频题目分类
**入门 (Easy)**
- Two Sum
- Valid Parentheses
- Reverse Linked List
- Binary Tree Traversal

**中等 (Medium)**
- Longest Substring Without Repeating
- 3Sum
- Merge Intervals
- Number of Islands

**困难 (Hard)**
- Median of Two Sorted Arrays
- Regular Expression Matching
- Trapping Rain Water

## 面试技巧

### 算法题
1. 确认理解问题
2. 给出暴力解法
3. 分析复杂度
4. 优化思路
5. 写代码并测试
6. 讨论边界情况

### 系统设计
1. 明确需求和约束
2. 估算容量 (QPS、存储)
3. 高层架构设计
4. 深入关键组件
5. 讨论权衡 (trade-off)

### 行为面试
- 准备项目故事 (STAR 法则)
- 团队协作经验
- 解决冲突能力
- 学习成长经历

## 面试清单

### 面试前
- [ ] 复习基础知识
- [ ] 刷 50-100 道算法题
- [ ] 准备项目介绍
- [ ] 了解公司技术栈

### 面试中
- 保持沟通
- 不懂不要装懂
- 展示思考过程
- 积极提问

### 面试后
- 总结经验
- 复习薄弱点
- 跟进感谢
