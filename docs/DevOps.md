# DevOps

## 概述
DevOps 是开发 (Dev) 和运维 (Ops) 的实践集合,旨在加速软件交付。

## 核心实践

### CI/CD (持续集成/持续部署)
- **GitHub Actions** - 最流行的 CI/CD 平台
- **GitLab CI** - 一体化 DevOps 平台
- **Jenkins** - 传统 CI/CD 工具

### 容器化
- **Docker** - 容器运行时
- **Kubernetes** - 容器编排
- **Docker Compose** - 本地多容器编排

### 基础设施即代码 (IaC)
- **Terraform** - 多云基础设施
- **Ansible** - 配置管理
- **Pulumi** - 用代码定义基础设施

### 监控与日志
- **Prometheus** - 指标收集
- **Grafana** - 可视化监控
- **ELK Stack** - 日志分析

## 开发者必备技能
1. **[版本控制](版本控制.md)** - Git 工作流
2. **Docker** - 本地开发环境
3. **CI/CD** - 自动化测试和部署
4. **Linux** - 服务器基础
5. **网络基础** - HTTP/DNS/SSL

## 自动化脚本
- [Python](Python.md) - 运维脚本
- [Go](Go.md) - 工具链开发
- Shell/Bash - 服务器操作

## DevOps 文化
- 自动化一切
- 基础设施不可变
- 监控驱动决策
- 持续改进

## 学习路径
1. Linux 基础
2. Docker 入门
3. CI/CD 配置
4. 监控体系
5. Kubernetes 进阶
