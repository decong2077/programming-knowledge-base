# Python

## 概述
Python 是一门优雅且强大的高级编程语言,以简洁易读著称。

## 核心特性
- 简洁的语法
- 动态类型系统
- 丰富的标准库
- 强大的第三方生态

## 应用领域
- **Web 开发**: FastAPI, Django, Flask (参见 [后端开发](后端开发.md))
- **数据分析**: Pandas, NumPy
- **人工智能**: PyTorch, TensorFlow
- **自动化脚本**: 日常任务自动化
- **爬虫**: Scrapy, BeautifulSoup
