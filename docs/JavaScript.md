# JavaScript

## 概述
JavaScript 是 Web 的核心语言,前后端通吃。

## 核心特性
- 动态类型
- 事件驱动
- 单线程异步 I/O
- 原型链继承

## 应用领域
- **前端开发**: DOM 操作、框架开发 (参见 [前端开发](前端开发.md))
- **后端开发**: Node.js、Express (参见 [后端开发](后端开发.md))
- **移动端**: React Native
- **桌面应用**: Electron

## 生态系统
- **包管理**: npm, yarn, pnpm
- **构建工具**: Vite, Webpack
- **测试框架**: Jest, Vitest (参见 [测试](测试.md))

## 学习路径
1. 基础语法和 DOM
2. 异步编程 (Promise/async-await)
3. 现代 JavaScript (ES6+)
4. 框架学习: React 或 Vue
