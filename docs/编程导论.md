# 编程导论

## 什么是编程
编程是告诉计算机如何执行任务的艺术。通过编程语言,我们将人类思维转化为机器可执行的指令。

## 核心概念
- **变量与数据类型** - 存储和管理信息
- **控制流** - 条件判断和循环
- **函数** - 代码复用的基本单元
- **数据结构** - 组织数据的方式
